Instructions

  • Created : Feb, 04, 2022
  • Last Updated: Feb, 04, 2022
  1. Checkout the dev_CFM branch from VAST.js
    https://github.com/imonology/VAST.js/tree/dev_CFM and install the node modules (npm install)
  2. Checkout the SPS branch from https://github.com/hebrecht/aedes/tree/SPS
    install the node modules (npm install)
  3. Install the local VAST.js module for aedes
    • Make sure you are in the aedes directory
    • npm install ${path to local VAST.js module} (npm install ../VAST.js)
  4. Ensure that the global_server of VAST.js is running
    • Make sure you are in the VAST.js directory
    • node test/global_server_starter.js
  5. Run the modified MQTT broker that is the gateway
    • Make sure you are in the aedes directory
    • node example/example_VAST_GW.js
  6. Run a test subscriber
    • Make sure you are in the aedes directory
    • node example/subscriber.js
  7. Run a test publisher
    • Make sure you are in the aedes directory
    • node example/publisher.js
