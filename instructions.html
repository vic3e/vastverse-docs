Instructions

  1. Checkout the dev_CFM branch from VAST.js
    https://github.com/imonology/VAST.js/tree/dev_CFM and install the node modules (npm install).
  2. Checkout the SPS branch
    https://github.com/hebrecht/aedes/tree/SPS and install the node modules (npm install)
  3. Install the local VAST.js module for aedes
    a.Make sure you are in the aedes directory.
    b.npm install ${path to local VAST.js module} (npm install ../VAST.js).
  4. Ensure that the global_server of VAST.js is running
    a.Make sure you are in the VAST.js directory
    b.node test/global_server_starter.js.
  5. Run the modified MQTT broker that is the gateway
    a.Make sure you are in the aedes directory
    b.node example/example_VAST_GW.js.
  6. Run a test subscriber
    a.Make sure you are in the aedes directory
    b.node example/subscriber.js.
  7. Run a test publisher
    a.Make sure you are in the aedes directory
    b.node example/publisher.js.
